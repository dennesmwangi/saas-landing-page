What channels does the API support?

Our API supports bulk messaging across three major communication channels:

  • SMS – Global bulk text messaging with high deliverability
  • WhatsApp – Official WhatsApp Business messaging for notifications and conversations
  • Email – High-volume transactional and marketing email delivery

Can I send bulk messages to thousands of users at once?

Yes. The API is built for high-volume messaging and supports sending to thousands (or even millions) of recipients in a single request using batch endpoints.

Does the API support message personalization?

Absolutely. You can personalize messages using dynamic variables such as:

  • First name
  • Order number
  • Appointment time
  • Custom fields from your database

For example:

Hi {{first_name}}, your order #{{order_id}} has been shipped.

How do I track delivery and engagement?

The API provides real-time status tracking and webhooks for:

  • Sent
  • Delivered
  • Read (WhatsApp & Email)
  • Opened (Email)
  • Clicked (Email links)
